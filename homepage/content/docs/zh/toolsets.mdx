# ToolSets（可加载工具集）

ToolSet 是一组“成套工具 + 默认使用说明”的组合。

当你加载一个 ToolSet 时，ShipMyAgent 会做两件事：

1. 把该 ToolSet 的 **默认说明**以 **system prompt** 注入（让模型立刻理解使用规范与边界）
2. 将该 ToolSet 的 **tools 一次性注册**到 Agent 的工具表中（后续即可直接调用这些工具）

## 发现可用 ToolSets

使用内置工具：

- `toolset_list`：列出可用 ToolSets，以及当前已加载的 ToolSets

## 加载 ToolSet

使用内置工具：

- `toolset_load`：按 `id/name` 加载 ToolSet

示例（加载联系人簿工具集）：

```json
{ "name": "contact_book" }
```

示例（加载 exec_shell，且允许覆盖同名工具）：

```json
{ "name": "exec_shell", "override": true }
```

## 常见内置 ToolSets

> 具体可用列表以 `toolset_list` 输出为准。

- `contact_book`：联系人簿相关（查找/列表/设置昵称/发送/删除/手动 upsert）
