# Built-in Skills（内置）

ShipMyAgent 内置了一些 Skills（SKILL.md），随包发布。

在执行 `sma init` 时，会把这些内置 skills 从安装包的 `bin/` 复制到你的用户目录 `~/.ship/skills`，之后就会像普通 skills 一样被发现与加载。

默认扫描 roots：

- 项目内：`<project>/.ship/skills`
- 用户目录：`~/.ship/skills`

使用方式与普通 skills 完全一致：

- 发现：`skills_list`
- 加载：`skills_load`
